== Description ==

It is a plugin that allows you to receive customer quotes on the PDP product detail page through WhatsApp, generating the meta information about the customer's product of interest.

== Installation ==


1. Upload `venduy-for-woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place, check in PDP in your templates
